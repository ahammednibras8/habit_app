## 🚀 Quick Start

### Prerequisites
```bash
flutter --version
# Flutter ^3.5.4
# Dart ^3.0.0
```

### Installation
```bash
# Clone repository
git clone [repository-url]

# Navigate to project
cd mindful_habit_builder_app

# Install dependencies
flutter pub get

# Run the app
flutter run
```
